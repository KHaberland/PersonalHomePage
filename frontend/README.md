# Frontend (Next.js)

Фронтенд будет создан в Фазе 2 плана разработки.

## Структура (после Фазы 2)

```
frontend/
├── app/
├── components/
├── lib/
├── locales/
└── styles/
```

## Локальная разработка

```powershell
# Установить зависимости
npm install

# ESLint
npm run lint

# Prettier
npm run format
```
